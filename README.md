# RegexUtils
# How do I use it?
- Add it in your root build.gradle at the end of repositories:
```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

- Add the dependency
```
	dependencies {
	        implementation 'com.github.AndroidRookie706:RegexUtils:Tag'
	}

```
